# Student Performance Analysis using PCA & Classification

## Problem Statement
Analyze student academic performance data to identify key factors affecting scores and predict career aspirations using machine learning techniques.

## Dataset
- **Source:** Student Scores Dataset
- **Records:** 1000+ students
- **Features:** Math, Physics, Chemistry, Biology, English, Geography scores + demographic info

## Tools & Libraries Used
- Python
- Pandas, NumPy
- Scikit-learn (PCA, Logistic Regression, StandardScaler)
- Matplotlib, Seaborn

## Project Workflow
1. Data Loading & Exploration
2. Data Cleaning & Preprocessing
3. Feature Scaling using StandardScaler
4. Dimensionality Reduction using PCA
5. Classification using Logistic Regression
6. Model Evaluation — Accuracy Score & Confusion Matrix

## Key Insights
1. PCA reduced dimensions while preserving maximum variance in data
2. Weekly self-study hours strongly correlates with higher scores
3. Students with part-time jobs show lower academic performance
4. Logistic Regression achieved strong accuracy on career prediction

## Results
- Successfully reduced high-dimensional score data using PCA
- Built classification model to predict student career aspirations
- Identified key factors affecting student performance
